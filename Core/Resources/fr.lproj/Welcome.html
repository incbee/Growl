Bienvenue dans Growl !

Growl est un système de notification que les autres applications peuvent utiliser, de manière unifiée, afin de fournir à l'utilisateur une gestion centralisée du contrôle de ces notifications.

Cette introduction s'affichera une fois par version majeure, afin d'indiquer les nouveautés, tandis que les quelques étapes simples de prise en main rapide s'afficheront une seule fois.
