Bem-vindo ao Growl!

Growl é um sistema de notificação que outras aplicações podem usar para exibir notificações de forma unificada, e para fornecer ao usuário um local centralizado, no qual é possível controlar estas notificações.

Esta introdução só será exibida uma vez a cada nova versão, para exibir informações sobre novidades da versão, e uma vez para alguns passos rápidos e simples para ajudá-lo nos primeiros passos.
