Что нового в Growl 2.0.1

- Исправлена ошибка позиционирования, появившаяся в 2.0, спасибо за отзывы!
- Исправлены многочисленные вылеты.

Изменения в Growl 2

Общее
- Добавлена поддержка центра уведомлений в OS X 10.8.
- Добавлена поддержка Prowl и Boxcar.
- Улучшено Окно Rollup.
- Growl теперь работает в песочнице.

Настройки
- Новые переключатели Вкл/Выкл стали проще в использовании.

Плагины
- Больше настроек голосового чтения.
- Плагины визуального отображение отделены от плагинов действий.
- Плагин "Музыкальное видео" теперь может выравнивать текст влево-вправо.
- Исправлены проблемы высоты с многострочными уведомлениями плагина "Музыкальное видео"
- "Нано" теперь может показываться в разных позициях слева, по центру и справа.
- Добавлен плагин Звуковое Действие.
- Добавлена возможность не использовать визуальные плагины вообще.

Исправления ошибок
- Исправлено чрезмерная нагрузка на процессор.
- Исправлены множественные падения.
- Сочетания клавиш теперь можно отменить.


Изменения Growl 1.4

Общее
- Изменен дизайн вкладки Приложения! Она стала легче и проще в использовании.
- Настройки окна истории перенесены на отдельную вкладку.
- Множественные улучшения в работе с сетью.

Улучшения панели настроек
- Новый слайдер включения/выключения.
- Поддержка клавиатуры в выборе позиций.
- Изменен дизайн вкладки Общие.
- Вкладка Сеть - Добавлена поддержка подписки
- Вкладка История, контекстное меню для таблицы истории.
- Вкладка Rollup с настройками окна истории сообщений
- Добавление нового плагина отображения больше не требует перезапуска Growl.

Работа с сетью
- Поддержка Notification-Callback-Target, URL поля в GNTP. После нажатия, Growl позволит OS X определить как открыть URL (например, браузером по-умолчанию для http:// ссылок)
- Поддержка подписок. Позволяет подписаться на все приложения на удаленном компьютере.
- Общие улучшения в сетевой аутентификации, убрано требование всегда использовать заголовок с паролем, если пароль не задан.
- Исправлены ошибки работы с сетью при разрешении адреса.
- Используется локальный сокет, если нет необходимости принимать входящие соединения.
- Использование кеширования адресов.

Applescript
- Поддержка кликов по URL, аналогичная Notification-Callback-Target и GNTP

Исправления ошибок
- Исправлено падение в плагине webkit.
- Исправлено падение при форварде.
- Исправлена проблема с черезмерным использованием процессора голосовыми уведомлениями.
- Исправлено падение с ошибкой CFErrorGetDomain для стилей WebKit.
- Исправлено дублирование списка плагинов при переустановке.
- Теперь учитывается и используется Notification-Callback-Target, даже при отсутствии заголовка метода.
- Исправлены проблемы с повреждением базы данных истории.
- Исправлены проблемы сортировки при удалении одиночных записей списка истории.
