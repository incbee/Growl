เปลี่ยนแปลงใน Growl 2.0.1

- จัดการกับปัญหาเรื่องตำแหน่งที่เกิดขึ้นใน 2.0 (ขอบคุณสำหรับคำติชม)
- แก้ปัญหาการปิดตัวเอง

สิ่งที่เปลี่ยนแปลงใน Growl 2

ทั่วไป
- รองรับศูนย์แจ้งของ OS X 10.8
- รองรับ Prowl และ Boxcar สำหรับการแสดงแจ้งเตือนบนจอ
- ปรับปรุงการตรวจสอบสถานะ Idle ไร้กิจกรรม ของ Rollup 

การตั้งค่า
- สวิตช์ เปิด/ปิด ได้รับการปรับปรุงให้เข้าใจง่ายขึ้น

การแสดงผล
- เพิ่มการควบคุมมากมาย สำหรับการใช้เสียงอ่าน
- แยกการแสดงผลแบบกิจกรรม ออกจากการแสดงผลบนจอ
- เลือกอักษรชิดซ้ายหรือขวาได้ สำหรับรูปแบบมิวสิควิดีโอ
- แก้ไขปัญหาความสูงของรูปแบบมิวสิควิดีโอ เมื่อข้อความมีหลายบรรทัด
- เพิ่มการแสดงผล แบบกิจกรรมเสียง
- เพิ่มตัวเลือกที่จะไม่แสดงแจ้งเตือนบนจอ

แก้ไขบั๊ก
- ปัญหาการใช้ CPU สูงเกิน
- แอพปิดตัวเองหลายครั้ง
- สามารถยกเลิกทางลัดได้


สิ่งที่เปลี่ยนแปลงใน Growl 1.4

ทั่วไป
- ดีไซน์แท๊บ Applications ใหม่หมด ทำให้ใช้ง่ายขึ้น
- แยกการควบคุม Rollup ไปแท็บใหม่
- เพิ่มประสิทธิภาพในด้านการทำงานบนเครือข่าย

ปรับปรุงหน้าต่างตั้งค่า
- สวิชต์ปิดเปิดเป็น slider แบบใหม่
- เพิ่มการเลือกตำแหน่งด้วยคีย์บอร์ด
- จัดวางแท็บ ทั่วไป แบบใหม่
- เพิ่มการรองรับ Subscription ในแท็บ เครือข่าย
- แท็บ ประวัติ เพิ่มเมนูเพื่อข้ามไปการตั้งค่าของแอพฯ/การแจ้งเตือน
- แท็บ Rollup ใหม่สำหรับจัดการหน้าต่าง Rollup
- การเพิ่มรูปแบบ display ใหม่ ไม่ต้องรีสตาร์ท Growl แล้ว

เครือข่าย
- รองรับ Notification-Callback-Target, เพิ่มช่อง URL ใน GNTP เมื่อคลิก Growl จะให้ OS X จัดการว่าจะเปิด URL อย่างไร (เช่น ใช้เบราเซอร์ในการเปิด http:// URL)
- รองรับการลงชื่อ Subscription เพื่อบอกรับการแจ้งเตือนจากแอพฯ จากเครื่องบนเครือข่าย
- การปรับปรุงทั่วไปในส่วนของลงชื่อเข้าใช้จากเครือข่าย ยกเลิกการต้องใช้รหัสผ่าน และใช้ headers ของรหัสผ่านถ้าไม่ได้กำหนดรหัสผ่าน
- แก้ไขการปิดเอง ที่เกี่ยวกับเครือข่าย และ address data resolution
- ผูกกับ local socket เท่านั้น เมื่อไม่ได้รับข้อความจากภายนอก
- จัดการเรื่องแคชข้อมูล และสังเกตการเปลี่ยนแปลงบนเครือข่ายเพื่อล้างแคช

AppleScript
- รองรับ feedback จากการคลิก URL เหมือนใน Notification-Callback-Target ใน GNTP

แก้ไขบั๊ก
- การปิดตัวเองเกี่ยวกับปลั๊กอิน WebKit
- การปิดตัวเองเมื่อส่งต่อข้อความ
- ปัญหาเกี่ยวกับการใช้ CPU สูงเกินเมื่อเลือกใช้เสียงอ่าน ขณะรอข้อความก่อนหน้า
- การปิดตัวเองเกี่ยวกับ CFErrorGetDomain สำหรับรูปแบบที่ใช้ WebKit
- ปัญหารายการซ้ำเมื่อลงปลั๊กอินอีกครั้ง
- Notification-Callback-Target ใช้โดยไม่จำเป็นต้องมี Method Header แล้ว
- แก้ขัดฐานข้อมูลเสียหาย
- ปัญหาจัดเรียงเมื่อลบรายการหนึ่งออกจากรายการประวัติ
